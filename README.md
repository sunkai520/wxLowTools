# wxLowTools
wx低版本登录修复工具

## 软件说明

本工具用于修复微信低版本登录验证码问题，支持扫码重新登录，解决验证码更新导致无法登录的情况。

## 使用步骤

**1. 下载软件**

下载 `wxLowTools.exe` 文件到本地，双击运行。

**2. 获取激活码**

软件启动后会显示激活界面：

- 复制界面中的 **开发者ID** 和 **本机机器码（MAC地址）**
- 将以上信息发送给作者，获取对应的激活码

> 如需试用，请加V联系作者

**3. 激活软件**

将作者提供的激活码粘贴到激活码输入框中，点击 **「立即激活」** 完成激活。

**4. 配置微信目录**

激活成功后，在软件设置中配置好本机的微信安装目录。

**5. 打开二维码界面**

配置完成后，打开软件的二维码界面。

**6. 修复并登录**

点击 **「开始修复」**，待验证码更新后，使用手机微信扫描界面中的二维码完成登录。
